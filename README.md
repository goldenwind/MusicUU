# 音乐间谍
搭建一个简单的音乐间谍第三方客户端

####使用到的开源项目
#####网络请求框架
[retrofit2](https://github.com/square/retrofit)  

#####json解析
[Gson](https://github.com/google/gson)  

#####视频播放
[Vitamio](https://github.com/yixia/VitamioBundle)  

#####图片加载
[picasso](https://github.com/square/picasso)
